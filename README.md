# SistemaCopo

Sistema web para control de requisiciones, inventario de almacén y expedientes de seguimiento.

## Módulos

- Login con Firebase Authentication.
- Dashboard principal.
- Pendientes / Requisiciones de compras.
- Inventario de almacén.
- Expedientes / Seguimiento con historial y documentos adjuntos.

## Tecnologías

- HTML, CSS y JavaScript.
- Firebase Authentication.
- Cloud Firestore.
- Firebase Storage.
- GitHub Pages.
